# Explore India Travels 🌍

A responsive, multi-page travel website showcasing beautiful Indian destinations, packages, and holiday experiences. Built as part of Assignment 1.

## 🚀 Live Demo
You can view the live deployed website here: 
https://venkatalikith-1705.github.io/Explore-India-Travels/

## 📂 Project Structure
This repository contains the following core files:
*   `index.html` - Home page featuring popular highlights and promotions.
*   `about.html` - Information about the travel agency, mission, and values.
*   `packages.html` - Available tour packages with detailed pricing layouts.
*   `gallery.html` - Interactive photo gallery displaying travel destinations.
*   `contact.html` - Contact details and a communication form.
*   `style.css` - Custom CSS stylesheet providing layout design, colors, grid/flexbox systems, and full responsiveness across mobile, tablet, and desktop viewports.
*   `script.js` - Dynamic JavaScript driving interactive features like mobile navigation toggle, dynamic package cost calculation, interactive gallery filtering, and frontend client form validation.

## ✨ Features Implemented
1. **Semantic HTML5 Layouts:** Fully structured pages utilizing modern structural markup.
2. **Responsive CSS Design:** Custom media queries ensuring the interface scales beautifully from smartphone screens to large desktops.
3. **SEO Optimized:** Meta tags and descriptive `alt` text configured across pages for discoverability and accessibility.
4. **Interactive JavaScript Controls:** Package pricing interactive calculation, form validation, and dynamic gallery filters.
